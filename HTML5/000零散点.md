1. 推出HTML5的原因  
    1. Web浏览器之间的兼容性低
    2. 文档结构不够明确
    3. Web应用程序的功能受到限制
2. HTML5语法的改变  
    1. DOCTYPE声明
    2. 指定字符编码
    3. 具有布尔值的属性
    4. 省略引号
3. 新增的元素和废除的元素
* 新增的结构元素
        section，article，header，footer，nav，figure
* 新增的其他元素
        video，audio，canvas
* 新增的Input元素类型
        email，url，number，range，Date Pickers
* 废除的元素
        不再使用frame框架，使用iframe
* 新增的全局属性  
    1. contentEditable属性
    2. designMode属性
    3. hidden属性
    4. spellcheck属性
    5. tabindex属性
4. 
````html
<!DOCTYPE html> <!-- 文档声明 -->
<html lang="cmn-hans">
<head>
        <meta> <!-- 文档元数据，附加信息-->
        <meta charset="UTF-8"> <!-- 指定网页内容编码 -->
        <meta name="viewport" content="width=device-width, initial-scale=1.0"> <!-- 适配移动端 -->
        <meta http-equiv="X-UA-Compatible" content="ie=edge"> <!-- IE内核切换 -->
</head>
<body>
</body>
</html>
````
