### 四类选择器
* 基本选择器  
    1. 通用选择器
    ````css
    *{

    }
    ````
    2. 标签选择器
    ````css
    div{

    }
    ````
    3. 类选择器
    ````css
    .class属性值{

    }
    ````
    4. id选择器
    ````css
    #id属性值{
        
    }
    ````
* 复合选择器  
    1. 多元选择器
    ````css
    div,p,span{

    }
    ````
    2. 后代元素选择器
    ````css
    div span{

    }
    ````
    3. 子元素选择器
    ````css
    div > span{

    }
    ````
    4. 相邻元素选择器
    ````css
    div + div{

    }
    ````
